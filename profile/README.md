What is this?

FuryMC Launcher is a modified version of Zalith Launcher 2.

Main goal:

- Keep offline/local login always enabled
- No forced restrictions
- Simple, usable launcher focused on accessibility

This is not trying to reinvent everything — just making ZL2 usable the way it should be.

Status

Development is currently paused.

Work will continue after the author's exams are finished.
After that, this project will be actively developed and maintained.
